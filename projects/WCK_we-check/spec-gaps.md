## 설계서 개요 — 위체크(We-check) 표준형 상품 설계서 v2.11

### 서비스 개요
- Web-view 기반 모바일 스크래핑 서비스 (비회원/회원 분기)
- 지원 기관: 정부24, 홈택스, 국민건강보험, 대법원 등
- 커스터마이징 등급: **표준형** (색상+로고만), **표준형고도화**, **맞춤형**

### 화면 도메인 구성

| 도메인 | 범위 | 비고 |
|--------|------|------|
| A | 위체크 메인, 약관동의 (A00~A03) | |
| B | 정보입력, 주소입력, 계좌목록 (B00~B11) | B01/B02 하나증권, B10/B11 삼성증권 |
| C | 인증 전체 — 인증서 선택·요청·완료, 캡챠 (C00~C40) | |
| E | 인증완료, 에러 전체 (E00~E40) | |
| T | 팝업 공통 — 정보입력 팝업, 주소검색, 공지 (T00~T23) | |

### 주요 정책 요약

| 항목 | 내용 |
|------|------|
| 프로그레스바 | `n/n` 형식, 10단계 이상은 `%` 표기, 최종 단계는 체크 아이콘, 1/1이면 미표출 |
| X 버튼 닫기 | Callback URL로 결과 전송 (V2.0~) |
| 세션 만료 | 비활동 5분 또는 간편인증 만료 시 E15 이동 |
| 스크래핑 로딩 | 진행률 `(완료 수 / 전체 수) × 100`, 첫 표출 최소 7%, 최대 99% |
| 캡챠 | 25/07 이후 신규 고객사부터 기본 적용 (기존 고객사 별도 적용) |
| 공지팝업(T13) | 어드민 관리, 공지 없으면 메인 바로 이동 (V2.0~) |

### V2.11 주요 변경사항
- **B01/B02** 주소정보입력 신규 (하나증권 전용)
- **B10/B11** 계좌목록조회 신규 (삼성증권 전용)
- **C12_2** 보안문자입력 정책삭제·화면최신화
- 복지로·장애인증명서 관련 화면 추가

### 에러 코드 분류

| 코드 | 유형 | 화면 |
|------|------|------|
| E31 | 스크래핑 기본 오류 | E40 (2버튼) |
| E32 | 문구 변형 오류 | E40 |
| E33 | 문구 변형 + 거래번호 | E40 |
| E34 | WC 코드 오류 (쿠콘 공통화면) | 별도 고정 디자인 |
| E02/E04 | 팝업형 오류 | T 도메인 팝업 |
