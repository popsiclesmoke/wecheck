## 화면 흐름 요약

**시나리오 1: 정상 발급 요청**
```
[SRCH-005] 결과 선택 → [ISSU-001] 인증번호 입력 → (정상)
→ [ISSU-003] 서류 업로드 → (업로드 완료) → 발급 요청 → RSLT 도메인
```

**시나리오 2: 인증번호 오류**
```
[ISSU-001] → (잘못된 번호) → [ISSU-002] 오류 안내
→ 재입력 → (정상) → [ISSU-003]
```

**시나리오 3: 업로드 방식 선택**
```
[ISSU-003] → (파일 첨부 탭) → [ISSU-004] 방식 선택 → 카메라/갤러리 등 → [ISSU-003] 파일 첨부됨
```
