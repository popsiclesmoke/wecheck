## 화면 설계

---

### ISSU-001 인증번호 입력

**파일명:** `BR00.인증번호입력.html`

**진입 조건**
- SRCH-005 검색 결과 선택 완료 후

**화면 구성**

| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| 상단 | StatusBar + MainHeader | WeCheck 로고, 닫기(X) 버튼 |
| 페이지 타이틀 | 제목 + 안내 설명 | `heading1` / `body2`. 발송된 채널(SMS 등) 안내 포함 |
| 본문 | 카카오 알림톡 이미지 (선택) + Input | 인증번호 6자리 입력. `placeholder`: "인증번호 입력" |
| 하단 | Button (Primary, 비활성) | "확인" — 인증번호 미입력 시 비활성 |

**액션 및 분기**

| 액션 | 조건 | 결과 |
|------|------|------|
| 인증번호 입력 | 올바른 번호 | ISSU-003 서류 업로드 이동 |
| 인증번호 입력 | 잘못된 번호 | ISSU-002 오류 상태 전환 |
| 확인 버튼 탭 | 미입력 | 반응 없음 (버튼 비활성) |

---

### ISSU-002 인증번호 입력 — 오류

**파일명:** `BR00_1.인증번호입력_오류.html`

**진입 조건**
- ISSU-001에서 잘못된 인증번호 입력 시

**화면 구성**

ISSU-001과 동일 레이아웃. 추가 요소:

| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| Input 하단 | 인라인 오류 메시지 | `error` 색상, "인증번호를 다시 확인해 주세요" 등 |
| Input | border 색상 | `error` 색상으로 강조 |
| 하단 | Button (Primary) | 재입력 후 활성화 |

**액션 및 분기**

| 액션 | 조건 | 결과 |
|------|------|------|
| 인증번호 재입력 후 확인 | 올바른 번호 | ISSU-003 이동 |
| 인증번호 재입력 후 확인 | 또 잘못된 번호 | ISSU-002 유지 (오류 메시지 갱신) |

### ISSU-003 서류 업로드

**파일명:** `BR01.파일첨부.html`

**진입 조건**
- ISSU-001 인증번호 확인 완료 후

**화면 구성**

| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| 상단 | StatusBar + MainHeader | 닫기(X) 버튼 |
| 페이지 타이틀 | 제목 + 안내 설명 | `body2`. 발급 서비스명 강조 표시 (`primary` 색상) |
| 본문 | 서류 업로드 목록 | 서류별 첨부 영역. 미첨부: 점선 박스 + "파일 첨부" 문구. 첨부 완료: 파일명 + 삭제 버튼 |
| 하단 | Button (Primary) | "발급 요청" — 필수 서류 모두 업로드 시 활성 |

**액션 및 분기**

| 액션 | 조건 | 결과 |
|------|------|------|
| 파일 첨부 영역 탭 | — | ISSU-004 업로드 방식 선택 |
| 파일 삭제 버튼 탭 | — | 첨부 파일 제거, 해당 서류 미첨부 상태로 전환 |
| 발급 요청 버튼 탭 | 필수 서류 모두 업로드 | RSLT-014 대기열 로딩 → RSLT-004 결과 조회 |
| 발급 요청 버튼 탭 | 오류 발생 | RSLT 오류 화면으로 이동 |

---

### ISSU-004 업로드 방식 선택

**파일명:** `BR01_1.파일첨부_업로드선택.html`

**진입 조건**

**화면 구성**

하단 선택 시트(Bottom Sheet) 또는 팝업 형태로 표시.

| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| 딤 배경 | Overlay Dim | 하단부터 표시 |
| 선택 시트 | Bottom Sheet | 업로드 방식 목록 |
| 항목 | 카메라 촬영 / 갤러리 선택 / 파일 탐색기 | 아이콘 + 레이블 |
| 하단 | 취소 버튼 | "취소" |

**액션 및 분기**

| 액션 | 조건 | 결과 |
|------|------|------|
| 방식 항목 탭 | — | 해당 방식으로 파일 선택, ISSU-003 복귀 + 파일 첨부됨 |
| 취소 탭 | — | 시트 닫힘, ISSU-003 유지 |
