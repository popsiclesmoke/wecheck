## 화면 흐름 요약

**시나리오 1: 인증 완료 → SRCH 진입**
```
[AUTH-011] 인증번호 입력 성공 → [RSLT-001] 인증 완료 → 확인 → [SRCH-001]
```

**시나리오 2: 발급 완료 결과 조회**
```
[ISSU-003] 발급 요청 → [RSLT-014] 대기열 로딩 → [RSLT-004] 증명서 결과 조회
```

**시나리오 3: 세션 만료**
```
[SRCH/ISSU 임의 화면] → (세션 만료) → [RSLT-012] 세션 만료 → 확인 → [MAIN-001]
```

**시나리오 4: 복구 가능 오류 (재시도)**
```
[임의 도메인] → (오류) → [RSLT-005/006/016] 오류 안내 → 재시도 → 직전 요청 재시도
```

**시나리오 5: 정보 조회 오류 (유형별)**
```
[SRCH/ISSU] → (조회 실패) → [RSLT-007~010] 유형별 안내 → 재시도 또는 MAIN 복귀
```
