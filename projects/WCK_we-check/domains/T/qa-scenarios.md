### SCN-CMN-001 공통 단일버튼 팝업 표시 및 닫힘

**목적**
> 공통 단일버튼 팝업이 정상적으로 표시되고 확인 버튼 탭 시 닫히는지 확인한다.

**사전 조건**
- 임의 도메인 화면에서 팝업 호출 트리거 발생

**테스트 단계**

| 단계 | 액션 | 기대 결과 |
|------|------|----------|
| 1 | 팝업 호출 트리거 발생 | CMN-001 팝업이 화면 중앙에 오버레이로 표시됨 |
| 2 | 팝업 외부 영역 탭 | 팝업 유지 (외부 탭 닫힘 없음) |
| 3 | 확인 버튼 탭 | 팝업 닫힘, 호출 도메인 화면으로 복귀 |

**예외 케이스**

| 케이스 | 액션 | 기대 결과 |
|--------|------|----------|
| 팝업 표시 중 뒤로가기 | Android 뒤로가기 | 팝업 닫힘 또는 비활성화 처리 (정책 확인 필요) |

### SCN-CMN-003 간편인증서 알림 미수신 팝업 — 재시도

**목적**
> 인증 알림 미수신 시 팝업이 표시되고 재시도 탭 시 인증 요청이 재발송되는지 확인한다.

**사전 조건**
- AUTH-009 (인증 대기 중) 화면 표시 상태

**테스트 단계**

| 단계 | 액션 | 기대 결과 |
|------|------|----------|
| 1 | "알림을 받지 못했나요?" 탭 | CMN-004 알림 미수신 팝업 표시 |
| 2 | 재시도 버튼 탭 | 팝업 닫힘, 인증 요청 재발송, AUTH-008 화면으로 이동 |
| 3 | 다시 팝업 열고 취소 탭 | 팝업 닫힘, AUTH-009 유지 |

**결과**
- [ ] 통과
- [x] 실패 — 사유: `T12.간편인증서_알림미수신팝업.html` 에서 팝업 우측 버튼 레이블이 "재시도"가 아닌 "인증서 재선택"으로 표시됨. design-spec(CMN-004) 명세는 "취소 / 재시도" 이나 HTML에는 "닫기 / 인증서 재선택"으로 구현되어 있어 버튼 레이블 불일치.

### SCN-CMN-004 공지 팝업 (1개) 표시 및 닫힘

**목적**
> 공지 1건일 때 단일 공지 팝업이 표시되고 닫힘 처리가 올바른지 확인한다.

**사전 조건**
- 서버에 공지 1건 등록 상태

**테스트 단계**

| 단계 | 액션 | 기대 결과 |
|------|------|----------|
| 1 | MAIN-001 진입 | CMN-005 공지 팝업 표시 |
| 2 | 팝업 내용 확인 | 제목·본문 정상 표시, 확인 버튼 표시 |
| 3 | 확인 버튼 탭 | 팝업 닫힘, TERM-001로 이동 |

**결과**
- [ ] 통과
- [x] 실패 — 사유: `T13.위체크메인_공지팝업_1개.html` 에서 닫기 버튼(`popup-close`)의 배경색이 `var(--color-bg-disabled)` (Secondary 스타일)로 구현됨. design-spec(CMN-005) 명세는 "Button (단일, Primary)"이나 Secondary 색상 적용.

### SCN-CMN-005 공지 팝업 (2개 이상) swipe 및 닫힘

**목적**
> 공지 2건 이상일 때 swipe로 공지를 전환하고 닫힘이 올바른지 확인한다.

**사전 조건**
- 서버에 공지 2건 이상 등록 상태

**테스트 단계**

| 단계 | 액션 | 기대 결과 |
|------|------|----------|
| 1 | MAIN-001 진입 | CMN-007 swipe 공지 팝업 표시, dot 인디케이터 표시 |
| 2 | 좌우 swipe | 다음/이전 공지로 전환, dot 인디케이터 변경 |
| 3 | 확인 버튼 탭 | 팝업 닫힘, TERM-001로 이동 |

**결과**
- [ ] 통과
- [x] 실패 — 사유: `T13_2.위체크메인_공지팝업_2개이상.html` 에서 (1) 닫기 버튼(`popup-close`)이 `var(--color-bg-disabled)` 색상으로 구현되어 Primary 명세 미준수. (2) dot indicator 구조(`page-dots`)는 정적으로 구현되어 있으나 swipe 전환을 위한 JS 인터랙션 로직이 없어 dot 인디케이터 동적 변경 불가. T13_1(스크롤 팝업) 역시 동일하게 닫기 버튼 Primary 미준수.

### SCN-CMN-006 주소 검색 — 정상 조회 및 선택

**목적**
> 공통 주소 검색에서 검색어 입력 후 결과 목록이 표시되고 항목 선택 시 주소가 반환되는지 확인한다.

**사전 조건**

| 3 | 결과 목록 항목 탭 | 주소 반환, 호출 도메인 화면으로 복귀 + 주소 자동 입력 |
