## 화면 흐름 요약

**시나리오 1: 정보 안내 팝업 (단일)**
```
[임의 도메인 화면] → (안내 필요) → [CMN-001] 공통 팝업 → 확인 → 원래 화면 복귀
```

**시나리오 2: 확인/취소 팝업**
```
[임의 도메인 화면] → (선택 필요) → [CMN-002] 투버튼 팝업 → 확인/취소 → 분기 처리
```

**시나리오 3: 알림 미수신**
```
[AUTH-009] 인증 대기 → [CMN-004] 알림 미수신 팝업 → 재시도 → [AUTH-008] 재요청
```

**시나리오 4: 공지 팝업**
```
[MAIN-001] 메인 → 공지 건수 분기
  → 1건·짧음: [CMN-005] → 확인 → [TERM-001]
  → 1건·긺:  [CMN-006] → 확인 → [TERM-001]
  → 2건+:    [CMN-007] swipe → 확인 → [TERM-001]
```

**시나리오 5: 공통 주소 검색**
```
[임의 도메인] → (주소 검색 호출) → [CMN-008] 주소 검색
  → 결과 없음: [CMN-009] → 재입력
  → 결과 있음: [CMN-010] 결과 목록 → 선택 → 주소 반환
```
