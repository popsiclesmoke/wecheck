## 공통 레이아웃 규칙

| 항목 | 값 |
|------|----|
| 화면 너비 | `360px` (`var(--screen-width)`) |
| 좌우 여백 | `20px` (`var(--page-margin)`) |
| 팝업 너비 | `290px` (중앙 정렬, `OverlayPopup`) |
| 팝업 버튼 높이 | `46px` (`var(--button-height)`) |
| 딤 배경 시작 | 헤더 하단(`top: 94px`)부터 |

> CMN 화면은 독립 페이지가 아닌 오버레이/컴포넌트로 동작한다. 전체 화면을 점유하지 않으며 호출 도메인 위에 레이어로 표시된다.

### CMN-008 공통 주소 검색

**파일명:** `T20.(공통)주소검색.html`

**진입 조건**
- 주소 입력이 필요한 도메인에서 주소 검색 호출 시 (SRCH 등)

**화면 구성**

| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| 상단 | StatusBar + MainHeader | WeCheck 로고, 닫기 버튼 |
| 검색 입력 | Input + 검색 아이콘 버튼 | 도로명·지번 주소 검색어 입력 |
| 본문 | 검색 전 안내 | "검색어를 입력하세요" 안내 문구 |

**액션 및 분기**

| 액션 | 조건 | 결과 |
|------|------|------|
| 검색어 입력 후 검색 | 결과 있음 | CMN-010 결과 목록 표시 |
| 검색어 입력 후 검색 | 결과 없음 | CMN-009 결과 없음 표시 |
