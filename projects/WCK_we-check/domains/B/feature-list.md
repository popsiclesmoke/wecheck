## 기능 목록

| Feature ID | 기능명 | 설명 | 인수 조건 | 선행 조건 | 비고 |
|------------|--------|------|-----------|-----------|------|
| F-SRCH-001 | 주소 정보 입력 | 증명서 발급을 위한 주소 정보를 입력한다. 주소 검색(CMN-008)을 통해 주소를 자동 입력할 수 있다. | 주소 입력 완료 시 다음 버튼 활성화 | AUTH 완료 (RSLT-001 경유) | CMN 주소검색 컴포넌트 연동 |
| F-SRCH-002 | 추가 정보 입력 | 주소 외 발급에 필요한 추가 항목을 입력한다. (기간, 용도 등 도입 기업 설정에 따라 상이) | 필수 항목 입력 완료 시 다음 버튼 활성화 | F-SRCH-001 | 항목 수는 도입 기업 설정에 따라 달라짐 |
| F-SRCH-003 | 검색 결과 목록 조회 및 선택 | 입력 정보 기반으로 조회된 결과 목록을 표시하고 항목을 선택한다. | 항목 선택 시 ISSU 도메인으로 이동 | F-SRCH-001 (또는 F-SRCH-002) | 라디오 버튼 형태 단일 선택 |
| F-SRCH-004 | 선택 변경 확인 | 이미 선택한 항목을 변경할 때 재확인 팝업을 표시한다. | 확인 시 선택 변경, 취소 시 기존 선택 유지 | F-SRCH-003 | CMN-002 투버튼 팝업 활용 |
