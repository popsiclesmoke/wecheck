## 화면 설계

---

### SRCH-001 주소 정보 입력

**파일명:** `B01.주소정보입력.html`

**진입 조건**
- RSLT-001 (인증 완료) 확인 버튼 탭 후

**화면 구성**

| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| 상단 | StatusBar + MainHeader | WeCheck 로고, 닫기(X) 버튼 |
| 페이지 타이틀 | 제목 텍스트 | `heading1` 크기 |
| 본문 | 주소 입력 섹션 | 입력 필드 레이블 + Input(readonly) + 검색 아이콘 버튼 |
| 하단 | Button (Primary, 비활성) | "다음" — 주소 미입력 시 비활성(disabled) |

**액션 및 분기**

| 액션 | 조건 | 결과 |
|------|------|------|
| 주소 검색 아이콘 탭 | — | CMN-008 공통 주소 검색 호출 |
| 주소 선택 완료 (CMN-010) | — | SRCH-003 전환 (주소 입력됨, 버튼 활성) |
| 추가 정보 입력 필요 | — | SRCH-002 전환 |

---

### SRCH-002 주소 정보 입력 — 추가 항목

**파일명:** `B01_1.주소정보입력_정보추가.html`

**진입 조건**
- SRCH-001에서 추가 항목 입력이 필요할 때 (또는 주소 입력 전 추가 항목 펼침)

**화면 구성**

| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| 상단 | StatusBar + MainHeader | |
| 페이지 타이틀 | 제목 텍스트 | |
| 본문 | 주소 입력 섹션 + 추가 항목 섹션 | 추가 입력 항목 표시 (기간, 용도 등) |
| 하단 | Button (Primary, 비활성) | 필수 항목 미완료 시 비활성 |

**액션 및 분기**

SRCH-001과 동일, 추가 항목 입력 완료 후 SRCH-004 전환.

### SRCH-004 주소 정보 입력 (활성화) — 추가 항목

### SRCH-005 검색 결과 목록 선택

**파일명:** `B10.검색_결과목록_선택.html`

**진입 조건**
- SRCH-003 또는 SRCH-004에서 다음 버튼 탭 후

**화면 구성**

| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| 상단 | StatusBar + MainHeader | 뒤로가기 버튼 포함 |
| 페이지 타이틀 | 제목 + 안내 텍스트 | |
| 본문 | 결과 카드 목록 | 각 항목: 라디오 버튼 + 결과 정보. 스크롤 가능 |
| 하단 | Button (Primary) | "선택 완료" |

**액션 및 분기**

| 액션 | 조건 | 결과 |
|------|------|------|
| 항목 탭 | 미선택 항목 | 해당 항목 선택, 라디오 활성 |
| 항목 탭 | 이미 선택된 다른 항목 있음 | SRCH-006 변경 확인 팝업 |
| 선택 완료 버튼 탭 | 항목 선택됨 | ISSU 도메인 진입 |

---

### SRCH-006 선택 변경 확인

**파일명:** `B11.검색_변경확인.html`

**진입 조건**
- SRCH-005에서 선택된 항목이 있는 상태에서 다른 항목 탭 시

**화면 구성**

CMN-002 (공통 투버튼 팝업) 패턴으로 구현. 화면 위에 오버레이로 표시.

| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| 딤 배경 | Overlay Dim | |
| 팝업 | OverlayPopup (290px) | "선택을 변경하시겠습니까?" 안내 |
| 하단 | Button 2개 | 취소(Secondary) · 확인(Primary) |

**액션 및 분기**

| 액션 | 조건 | 결과 |
|------|------|------|
| 확인 탭 | — | 선택 변경, 팝업 닫힘 |
| 취소 탭 | — | 기존 선택 유지, 팝업 닫힘 |
