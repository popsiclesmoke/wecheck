## 도메인 목적

본인인증(AUTH) 완료 후 증명서 발급을 위한 정보를 입력하고 검색 결과를 선택한다.
사용자가 주소 등 발급 기준 정보를 입력하면 해당 조건에 맞는 결과 목록이 조회되며, 원하는 항목을 선택해 발급 단계(ISSU 도메인)로 진입한다.

---

## 주요 기능 범위

- 주소 정보 입력 (도로명·지번 기반)
- 추가 발급 정보 입력 (기간, 용도 등 항목별 선택)
- 검색 결과 목록 조회 및 항목 선택
- 선택 변경 확인 (이미 선택한 항목 변경 시 재확인)

## 타 도메인 의존 관계

| 관계 | 도메인 | 설명 |
|------|--------|------|
| 선행 | AUTH | 본인인증 완료 후 SRCH 진입 (RSLT-001 인증완료 화면 경유) |
| 후행 | ISSU | 검색 결과 선택 완료 후 발급 단계로 이동 |
| 공유 | CMN | 주소 검색 시 CMN 공통 주소 검색 컴포넌트(CMN-008~010) 호출 |

---

## 미확정 사항

- 검색 결과 항목 표시 정보 → **확정(2026-06-10)**: B10 화면 기준, 결과는 **계좌 목록** 형태 — 항목별 라디오 + 이름 + "계좌" 배지 + 은행명 + 마스킹 계좌번호(예: "국민은행 123412231*****"). 기관명·코드·로고는 미표시. ※ 강다현 재확인 권장
- 추가 입력 항목의 유형 및 필수/선택 여부 (도입 기업별 상이할 수 있음)
- 결과 0건 시 처리 정책 (CMN-009 주소 검색 결과 없음 연계 또는 별도 안내)
