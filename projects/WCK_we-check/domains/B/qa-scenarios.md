### SCN-SRCH-001 주소 입력 → 검색 결과 선택 → ISSU 진입

**목적**
> SRCH 도메인 기본 흐름 전체가 정상 동작하는지 확인한다.

**사전 조건**
- AUTH 완료 (RSLT-001 인증완료 화면 표시 후 확인)

**테스트 단계**

| 단계 | 액션 | 기대 결과 |
|------|------|----------|
| 1 | SRCH-001 표시 | 주소 입력 화면 표시, 다음 버튼 비활성 |
| 2 | 주소 검색 아이콘 탭 | CMN-008 주소 검색 화면 호출 |
| 3 | 검색어 입력 후 결과 탭 | 주소 자동 입력, SRCH-003 전환, 다음 버튼 활성 |
| 4 | 다음 버튼 탭 | SRCH-005 검색 결과 목록 표시 |
| 5 | 결과 항목 탭 후 선택 완료 | ISSU 도메인 진입 |

| 4 | 결과 선택 완료 | ISSU 도메인 진입 |

### SCN-SRCH-004 선택 변경 확인 — 확인/취소 분기

**목적**
> 결과 목록에서 이미 선택된 항목이 있을 때 다른 항목 탭 시 변경 확인 팝업이 올바르게 동작하는지 확인한다.

**사전 조건**
- SRCH-005에서 항목 1개 선택 완료 상태

**테스트 단계**

| 단계 | 액션 | 기대 결과 |
|------|------|----------|
| 1 | 다른 항목 탭 | SRCH-006 변경 확인 팝업 표시 |
| 2 | 확인 탭 | 팝업 닫힘, 선택 변경됨 |
| 3 | 다시 또 다른 항목 탭 후 취소 탭 | 팝업 닫힘, 기존 선택 유지 |

**결과**

### SCN-SRCH-005 공통 주소 검색 연동 확인

**목적**
> CMN 공통 주소 검색 컴포넌트(CMN-008~010)가 SRCH 도메인과 올바르게 연동되는지 확인한다.

**테스트 단계**

| 단계 | 액션 | 기대 결과 |
|------|------|----------|
| 1 | SRCH-001에서 주소 검색 아이콘 탭 | CMN-008 표시 |
| 2 | 존재하지 않는 주소 검색 | CMN-009 결과 없음 표시 |
| 3 | 유효한 주소 검색 후 결과 탭 | SRCH-001/003 화면 복귀, 주소 자동 입력 확인 |
