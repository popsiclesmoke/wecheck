## 화면 흐름 요약

**시나리오 1: 기본 흐름 (추가 항목 없음)**
```
[RSLT-001] 인증완료 → [SRCH-001] 주소 입력 → (주소 입력 완료)
→ [SRCH-003] 활성화 → (다음) → [SRCH-005] 결과 목록 선택 → ISSU 도메인
```

**시나리오 2: 추가 항목 입력 포함**
```
[SRCH-001] → (추가 항목 펼침) → [SRCH-002] 추가 항목 입력
→ (주소 + 추가 항목 입력 완료) → [SRCH-004] 활성화 → [SRCH-005] → ISSU
```

**시나리오 3: 선택 변경**
```
[SRCH-005] 항목 선택됨 → (다른 항목 선택 시도) → [SRCH-006] 변경 확인
→ 확인: 선택 변경 → ISSU / 취소: 기존 선택 유지
```

**시나리오 4: 주소 검색 호출**
```
[SRCH-001/002] 주소 검색 아이콘 탭 → [CMN-008] 공통 주소 검색
→ [CMN-010] 결과 선택 → 주소 자동 입력, SRCH-003/004로 전환
```
