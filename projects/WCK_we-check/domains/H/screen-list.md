## 화면 흐름 요약

**시나리오 1: 이력 조회 → 상세 확인 (이력 있음)**
```
[MAIN-001] 메인 → (발급이력 진입 메뉴 탭)
→ [HIST-001] 발급 이력 목록 → (항목 선택)
→ [HIST-003] 발급 이력 상세 → (뒤로가기) → [HIST-001] 복귀
```

**시나리오 2: 이력 없음 (빈 상태)**
```
[MAIN-001] 메인 → (발급이력 진입 메뉴 탭, 이력 0건)
→ [HIST-002] 발급 이력 목록 — 이력 없음
```

**시나리오 3: 발급 완료 후 이력 연결 (선행 ISSU 연계)**
```
[ISSU/RSLT 발급 완료] → (발급 이력 보기) → [HIST-001] 발급 이력 목록
```
