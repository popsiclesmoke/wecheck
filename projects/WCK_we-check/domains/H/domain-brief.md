## 도메인 목적

과거에 발급한 증명서 이력을 목록으로 조회하고, 개별 발급 건의 상세 내역을 확인한다.
MAIN 화면의 발급이력 진입 메뉴를 통해 인증 세션 없이도 직접 진입할 수 있다.

---

## 주요 기능 범위

- 발급 이력 목록 조회 (발급일시·기관명·증명서명·상태 표시)
- 발급 이력 상세 확인 (개별 발급 건의 세부 정보)
- 이력 없음 상태 처리 (빈 상태 화면)

---

## 타 도메인 의존 관계

| 관계 | 도메인 | 설명 |
|------|--------|------|
| 선행 | MAIN | 메인 발급이력 진입 메뉴에서 진입 |
| 선행 | ISSU | 발급 완료 후 이력 화면으로 연결 가능 |
| 후행 | — | 없음 (이력 확인 후 종료) |

---

## 미확정 사항

- 미인증 사용자의 이력 진입 시 처리 정책 (재인증 유도 여부)
- 이력 조회 기간 범위 (전체 / 최근 N개월)
