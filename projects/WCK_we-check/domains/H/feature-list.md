## 기능 목록

| Feature ID | 기능명 | 설명 | 인수 조건 | 선행 조건 | 비고 |
|------------|--------|------|-----------|-----------|------|
| F-HIST-001 | 발급 이력 목록 조회 | 과거 발급 이력을 목록으로 조회한다. 각 항목에 발급일시·기관명·증명서명·발급 상태를 표시한다. | 발급 이력이 1건 이상 있을 때 목록이 정상 노출됨 | MAIN-001 발급이력 진입 메뉴 진입 | 인증 세션 없이 메인에서 직접 진입 가능 |
| F-HIST-002 | 발급 이력 상세 확인 | 목록에서 개별 발급 건을 선택하여 세부 정보(증명서명·기관명·발급일시·상태·발급 정보)를 확인한다. | 목록 항목 선택 시 해당 건의 상세 정보가 정상 노출됨 | F-HIST-001 | — |
| F-HIST-003 | 이력 없음 상태 처리 | 발급 이력이 한 건도 없는 경우 빈 상태(Empty State) 화면을 표시한다. | 발급 이력이 0건일 때 빈 상태 안내가 노출됨 | MAIN-001 발급이력 진입 메뉴 진입 | 일러스트/안내 문구 표시 |
