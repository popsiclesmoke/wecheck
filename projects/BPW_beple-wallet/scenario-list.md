## 시나리오 상세

### SCN-BPW-001: 내국인 온보딩
**분류:** Happy Path
**설명:** 내국인 사용자가 앱 최초 실행 후 언어 선택 → SMS 인증 → 약관 동의 → PIN 설정 → 생체인증 등록까지 완료하고 홈에 진입한다.
**화면 흐름:**
```
ONB-001: 언어 선택
  → ONB-002: 사용자 유형 선택 (내국인)
    → ONB-003: SMS 인증
      → ONB-004: 약관 동의
        → ONB-005: 가입 완료
          → ONB-011: PIN 설정
            → ONB-012: PIN 확인
              → ONB-013: 생체인증 등록
                → HOM-002: 홈 (Empty)
```

---

### SCN-BPW-002: 외국인 온보딩 (KYC Quick)
**분류:** Happy Path
**설명:** 외국인 사용자가 연락처 인증 → 약관 동의 → 여권 OCR + 안면 인식 KYC를 거쳐 계정을 생성하고 홈에 진입한다.
**화면 흐름:**
```
ONB-001: 언어 선택
  → ONB-002: 사용자 유형 선택 (외국인)
    → ONB-006: 연락처 인증
      → ONB-004: 약관 동의
        → ONB-007: KYC 방법 선택 (Quick)
          → ONB-008: 여권 OCR
            → ONB-009: 안면 인식
              → ONB-010: KYC 완료
                → ONB-011: PIN 설정
                  → ONB-012: PIN 확인
                    → ONB-013: 생체인증 등록
                      → HOM-002: 홈 (Empty)
```

---

### SCN-BPW-003: QR 결제 (정상)
**분류:** Happy Path
**설명:** 홈에서 결제 버튼 → QR 스캔 → 금액 확인 + PIN 인증 → 결제 완료 후 홈으로 복귀한다.
**화면 흐름:**
```
HOM-001: 홈 (결제 버튼)
  → PAY-001: QR 스캔
    → PAY-002: 결제 확인 + PIN
      → PAY-003: 결제 완료
        → HOM-001: 홈
```

---

### SCN-BPW-004: QR 결제 (잔액 부족)
**분류:** 분기 (잔액 부족 시)
**설명:** 결제 시도 중 잔액이 부족하면 충전 유도 화면을 거쳐 충전 도메인으로 이동한다.
**화면 흐름:**
```
HOM-001: 홈 (결제 버튼)
  → PAY-001: QR 스캔
    → PAY-002: 결제 확인
      → PAY-004: 잔액 부족 (Modal)
        → CHG-001: 충전 방법 선택
```

---

### SCN-BPW-005: Korbit 충전 (최초 연결)
**분류:** Happy Path
**설명:** Korbit 미연결 상태에서 Wallet Connect 연결 → 출금 동의 → 코인 매도 → 충전 완료까지의 전체 흐름이다.
**화면 흐름:**
```
HOM-001: 홈 (충전 버튼)
  → CHG-001: 충전 방법 선택 (Korbit)
    → CHG-KRB-001: Korbit 연결
      → CHG-KRB-002: 출금 동의 (1원 인증 + ARS)
        → CHG-KRB-003: 코인 선택
          → CHG-KRB-004: 수량 입력
            → CHG-KRB-005: 매도 확인
              → CHG-KRB-006: 공동인증
                → CHG-KRB-007: 충전 완료
                  → HOM-001: 홈
```

---

### SCN-BPW-006: Korbit 충전 (재충전)
**분류:** Happy Path
**설명:** Korbit 이미 연결된 상태에서 코인 선택부터 시작하여 매도 → 충전 완료한다.
**화면 흐름:**
```
HOM-001: 홈 (충전 버튼)
  → CHG-001: 충전 방법 선택 (Korbit)
    → CHG-KRB-003: 코인 선택
      → CHG-KRB-004: 수량 입력
        → CHG-KRB-005: 매도 확인
          → CHG-KRB-006: 공동인증
            → CHG-KRB-007: 충전 완료
              → HOM-001: 홈
```

---

### SCN-BPW-007: 직전송 충전
**분류:** Happy Path
**설명:** 외부 지갑에서 코인을 직접 전송하여 Beple Money로 충전한다.
**화면 흐름:**
```
HOM-001: 홈 (충전 버튼)
  → CHG-001: 충전 방법 선택 (직전송)
    → CHG-DRT-001: 직전송 안내
      → CHG-DRT-002: 코인/네트워크 선택
        → CHG-DRT-003: 환불주소 등록
          → CHG-DRT-004: 수신주소 안내 (QR)
            → CHG-DRT-005: 블록체인 확인 대기
              → CHG-DRT-006: 직전송 충전 완료
                → HOM-001: 홈
```

---

### SCN-BPW-008: 은행계좌 충전
**분류:** Happy Path
**설명:** 등록된 은행계좌에서 금액을 입력하고 PIN 인증 후 Beple Money를 충전한다.
**화면 흐름:**
```
HOM-001: 홈 (충전 버튼)
  → CHG-001: 충전 방법 선택 (은행계좌)
    → CHG-BNK-001: 계좌 선택 · 금액 입력 (PIN 오버레이)
      → CHG-BNK-002: 충전 완료
        → HOM-001: 홈
```

---

### SCN-BPW-009: ATM 출금
**분류:** Happy Path
**설명:** ATM에서 QR 스캔 → 금액 입력 → PIN 인증 → 현금 출금 완료한다.
**화면 흐름:**
```
HOM-001: 홈 (출금 버튼)
  → WDR-001: ATM QR 스캔
    → WDR-002: 출금 금액 입력
      → WDR-003: 출금 확인 + PIN
        → WDR-004: 출금 완료
          → HOM-001: 홈
```

---

### SCN-BPW-010: 은행계좌 등록
**분류:** Happy Path
**설명:** 홈 하단 탭바에서 계좌관리로 진입하여 은행 선택 → 계좌번호 입력 → ARS 인증으로 출금 계좌를 등록한다.
**화면 흐름:**
```
HOM-001: 홈 (탭바 > 계좌관리)
  → ACT-001: 계좌 목록
    → ACT-002: 계좌 등록
      → ACT-001: 계좌 목록 (등록 완료)
```

---

### SCN-BPW-011: 거래내역 조회
**분류:** Happy Path
**설명:** 홈 하단 탭바에서 거래내역으로 진입하여 개별 거래를 선택하고 상세를 확인한다.
**화면 흐름:**
```
HOM-001: 홈 (탭바 > 거래내역)
  → ACT-003: 거래 내역
    → ACT-004: 거래 상세
```

**화면 흐름:**
```
HOM-001: 홈 (탭바 > 설정)
  → SET-001: 설정 메뉴 (언어 변경)

      → SET-001: 설정 메뉴
```

### SCN-BPW-013: PIN 변경

**설명:** 홈 하단 탭바에서 설정으로 진입하여 현재 PIN 확인 후 새 PIN을 설정한다.
**화면 흐름:**
```
HOM-001: 홈 (탭바 > 설정)
  → SET-001: 설정 메뉴 (PIN 변경)
    → SET-003: PIN 변경
      → SET-001: 설정 메뉴

### SCN-BPW-014: 로그아웃
**분류:** Happy Path
**설명:** 홈 하단 탭바에서 설정으로 진입하여 로그아웃 후 앱 초기 화면으로 돌아간다.
**화면 흐름:**
```
HOM-001: 홈 (탭바 > 설정)
  → SET-001: 설정 메뉴 (로그아웃)
    → SET-001: 로그아웃 확인 Modal (취소/로그아웃)
      → ONB-001: 언어 선택
