## 화면 설계

---

### [SET-001] 설정 메뉴

**파일명:** `set-menu.html`

**화면 구성**

| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| 상단 | TopBar | 타이틀: "설정" |
| 본문 | 사용자 정보 카드 | 이름/이메일 또는 전화번호 표시 |
| 본문 | 메뉴 리스트 | 언어 설정 / 알림 설정(토글) / 생체인증(토글) / PIN 변경 / 로그아웃 |
| 각 항목 | 아이콘 + 라벨 + 우측 ">" 또는 토글 | — |

**동작 — 로그아웃 확인 Modal**

- "로그아웃" 메뉴 항목 탭 시 확인 Modal이 화면 위에 표시된다.
- Modal 구성:
  - 반투명 배경(dimmed), 중앙 카드
  - 타이틀: "로그아웃"
  - 본문: "정말 로그아웃 하시겠습니까?"
  - 버튼 영역: "취소" (Secondary, 좌측) / "로그아웃" (Danger, 우측)
- "취소" 탭 시 Modal 닫힘, 설정 화면 유지
- "로그아웃" 탭 시 세션 종료 후 ONB-001(언어 선택)으로 이동
