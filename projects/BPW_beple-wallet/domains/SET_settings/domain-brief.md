## 도메인 목적

앱 전반적인 사용자 환경 설정(언어, 알림, 생체인증, PIN 변경, 로그아웃)을 관리하는 설정 도메인이다.

---

## 주요 기능 범위

- 설정 메뉴 목록
- 언어 변경 (5개국어)
- 알림 설정 토글
- 생체인증 ON/OFF
- PIN 변경 (현재 PIN 확인 → 새 PIN 입력)
- 로그아웃
