## 도메인 목적

신규 사용자가 비플월렛을 처음 시작할 때 언어 선택, 회원가입, KYC 인증, PIN·생체인증 설정까지 완료하여 서비스를 사용할 수 있는 상태로 만든다.
내국인/외국인 두 유형을 분기 처리하여 각각에 최적화된 가입 경험을 제공한다.

---

## 주요 기능 범위

- 5개 언어 선택 (한국어, English, 中文, 日本語, Español) 및 전역 적용
- 내국인/외국인 사용자 유형 선택
- 내국인: 휴대폰 SMS 인증 → 약관 동의 → 가입 완료
- 외국인: 약관 동의 → 연락처 인증 → KYC Quick (여권 OCR + 안면인식 + HiKorea)
- 6자리 PIN 설정 및 확인
- 생체인증 등록 (선택)

---

## 사용자 유형별 분기

| 사용자 유형 | 분기 설명 |
|------------|---------|
| 내국인 | SMS 인증 → 약관 동의 → 완료 → PIN → 생체인증 |
| 외국인 | 약관 동의 → 연락처 인증 → KYC Quick → 완료 → PIN → 생체인증 |
