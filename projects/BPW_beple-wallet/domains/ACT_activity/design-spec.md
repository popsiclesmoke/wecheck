## 화면 설계

---

### [ACT-001] 계좌 목록

**파일명:** `act-accounts.html`

**화면 구성**

| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| 상단 | TopBar (뒤로가기) | 타이틀: "계좌 관리" |
| 본문 | 계좌 리스트 | 은행 로고 + 은행명 + 계좌번호(마스킹) + 삭제 버튼 |
| 본문 (Empty) | Empty 상태 | "등록된 계좌가 없습니다" + 등록 유도 텍스트 |
| 하단 | Button XLarge (Primary) | "계좌 등록하기" |

**화면 구성**

| 영역 | 컴포넌트 | 설명 |

### [ACT-003] 거래 내역

**파일명:** `act-history.html`

**화면 구성**

| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| 상단 | TopBar (뒤로가기) | 타이틀: "거래 내역" |
| 본문 | 필터 바 | 날짜 범위 / 유형(충전·결제·출금) / 상태 칩 |
| 본문 | 날짜별 그룹 리스트 | 날짜 헤더 + 거래 항목 |
| 각 항목 | 아이콘 + 거래명 + 금액 + 상태 | — |
| 본문 (Empty) | Empty 상태 | "거래 내역이 없습니다" |

### [ACT-004] 거래 상세
