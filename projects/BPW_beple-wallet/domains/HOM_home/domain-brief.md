## 도메인 목적

온보딩 완료 후 진입하는 메인 허브 화면으로, Beple Money(원화) 잔액과 퀵액션(충전·결제·출금), 배너를 제공하여 사용자가 핵심 기능으로 빠르게 접근할 수 있게 한다.

---

## 주요 기능 범위

- Beple Money(KRW) 잔액 표시
- Korbit 코인 자산 요약 표시
- 퀵액션 버튼: 충전 / 결제 / 출금
- 프로모션·공지 배너 표시 및 상세 이동
- 잔액 없음 Empty 상태 처리

---

## 사용자 유형별 분기

| 사용자 유형 | 분기 설명 |
|------------|---------|
| 자산 있음 | 잔액·코인 자산 표시 |
| 자산 없음 | Empty 상태 UI, 충전 유도 CTA |
