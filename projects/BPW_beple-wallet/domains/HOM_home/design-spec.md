## 화면 설계

---

### [HOM-001] 메인 홈

**파일명:** `hom-main.html`

**화면 구성**

| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| 상단 | TopBar (로고 + 알림 아이콘) | 비플월렛 로고 좌측, 알림 아이콘 우측 |
| 본문 | 잔액 카드 | Beple Money 잔액(KRW 원화), Primary/100 배경 카드 |
| 본문 | 퀵액션 버튼 3종 | 충전 / 결제 / 출금 아이콘+텍스트 버튼 가로 배열 |
| 본문 | 충전 수단 카드 | 연결된 충전 수단 상태 표시. Korbit(연결됨→코인매도 바로가기, 미연결→"연결하기" CTA), 은행계좌(등록됨→은행명/계좌마스킹+계좌충전 바로가기, 미등록→"계좌 등록하기" CTA), 외부지갑("USDT/USDC 직전송" 바로가기). 카드 형태, 각 항목에 아이콘+상태+액션 |
| 본문 | 배너 슬라이드 | 자동 슬라이드, 페이지 인디케이터 |
| 하단 | 탭바 | 3탭 (결제내역·홈·더보기), 홈 탭 선택 상태(센터 강조). 결제내역→act-history, 더보기→set-menu 연결 |
