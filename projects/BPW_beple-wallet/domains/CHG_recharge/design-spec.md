### [CHG-BNK-001] 계좌 선택 · 금액 입력

**파일명:** `chg-bnk-select.html`

**화면 구성**

| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| 상단 | TopBar (뒤로가기) | 타이틀: "계좌 충전" |
| 본문 | 계좌 선택 드롭다운 | 등록된 계좌 목록 (은행명 + 계좌번호 마스킹) |
| 본문 | 금액 입력 | 원화 금액 입력 + 빠른 금액 버튼 (+1만/+5만/+10만) |
| 본문 | 충전 한도 안내 | KYC 등급별 잔여 한도 표시 |
| 하단 | Button XLarge (Primary) | "충전하기" |

**동작 — PIN 인증 오버레이**

- "충전하기" 버튼 탭 시 PIN 6자리 입력 오버레이가 화면 위에 표시된다.
- 오버레이 구성: 반투명 배경(dimmed) + PIN 입력 패널 (타이틀: "PIN을 입력해주세요", PIN 도트 인디케이터 6자리, 숫자 키패드, 닫기 버튼)
- PIN 인증 성공 시 CHG-BNK-002(은행 충전 완료)로 이동
- PIN 5회 실패 시 오류 안내 후 화면 유지
