## 도메인 목적

사용자가 Beple Money(KRW)를 충전할 수 있는 3가지 채널(Korbit 코인매도 / 외부지갑 직전송 / 은행계좌)을 제공한다.
각 채널의 복잡한 외부 연동 플로우를 안전하게 처리하고, 완료까지 사용자를 안내한다.

---

## 주요 기능 범위

- 충전 방법 선택 화면 (3채널)
- **Korbit 채널**: PIN 인증 → Korbit Wallet Connect → 출금동의(1원 인증+ARS) → 코인 선택/수량 → 시장가 매도 → 공동인증 → 출금처리 → 완료
- **직전송 채널**: 안내 → 코인/네트워크 선택 → 환불주소 등록 → 수신주소 제공 → 블록체인 확인 대기
- **은행계좌 채널**: 계좌 선택 → 금액 입력 → PIN 인증 → 완료
- 충전 한도 초과 안내 (KYC Quick 50만원)

## 사용자 유형별 분기

| 사용자 유형 | 분기 설명 |
|------------|---------|
| Korbit 미연동 | Wallet Connect 연결 단계 포함 |
| Korbit 연동 완료 | 코인 선택부터 시작 |
| 은행계좌 미등록 | 계좌 등록 화면(ACT 도메인)으로 이동 유도 |

## 미확정 사항

- Beple Money 보유 한도 적용 기준 (내국인/외국인, KYC 티어별 분리 정책 미확정)
- 직전송 입력금액 vs 실제 전송금액 불일치 처리 (Triple-A API 정책 확인 필요)
- 은행계좌 충전 프로세스 재활용 여부 (기존 비플페이 오픈뱅킹 vs 신규 구축)
