## 1. 프로젝트 개요

### 1.1 배경 및 목적

비플월렛은 사용자가 보유한 코인 자산을 국내 가맹점에서 자유롭게 사용할 수 있게 하는 모바일 디지털 결제 지갑 서비스다.
Korbit 거래소 연동 및 외부 지갑 직접 전송을 통해 코인을 원화(Beple Money)로 환전하고, QR 결제·ATM 출금으로 소비할 수 있다.
기존 비플페이의 Beple Money·복지포인트·식권 기능을 통합하여, 하나의 지갑에서 모든 결제수단을 사용한다.

### 1.2 타겟 사용자

| 사용자 유형 | 설명 |
|------------|------|
| 내국인 코인 보유자 | Korbit 연동 또는 은행 계좌를 통해 충전 후 가맹점 결제 |
| 외국인 거주자·관광객 | 해외 지갑에서 USDT/USDC 직전송 → 원화 환전 → 국내 결제·ATM 출금 |

### 1.3 핵심 문제 정의

1. 코인 보유자가 국내 가맹점에서 코인을 직접 사용할 수 없다
2. 외국인(관광객·거주자)이 국내 카드/계좌 없이 결제 수단을 확보하기 어렵다
3. 기존 비플페이 Beple Money·복지포인트·식권이 별개 앱으로 분산되어 있다

## 2. 서비스 범위

### 2.1 포함 범위 (In-Scope)

| 도메인 | 기능명 | 한 줄 설명 |
|--------|--------|-----------|
| ONB | 온보딩 | 언어 선택, 내국인/외국인 분기, 회원가입, KYC, PIN·생체인증 설정 |
| HOM | 홈 | Beple Money 잔액 대시보드, 퀵액션, 배너 |
| CHG | 충전 | Korbit 코인매도, 외부지갑 직전송, 은행계좌 KRW 충전 |
| PAY | 결제 | QR 결제, Beple Money·복지포인트·식권 결제 |
| WDR | 출금 | ATM QR 코드 현금 출금 |
| ACT | 내역·계좌 | 계좌 관리, 거래 내역 조회 |
| SET | 설정 | 언어, 알림, 생체인증, PIN 변경, 로그아웃 |

### 2.2 제외 범위 (Out-of-Scope)

- 코인 매매·투자 기능
- 해외 법정화폐 송금
- Korbit 외 거래소 연동 (v1)
- 가맹점 관리 시스템
- 온라인 쇼핑 결제

## 4. 핵심 기능 요약

| 번호 | 기능명 | 설명 |
|------|--------|------|
| 1 | 내국인 회원가입 | SMS 인증 → 약관 동의 → 6자리 PIN → 생체인증(선택) |
| 2 | 외국인 KYC (Quick) | 여권 OCR → 안면 인식 → HiKorea 조회 → 50만원 한도 |
| 3 | Korbit 코인 충전 | Wallet Connect → 코인 선택·수량 입력 → 시장가 매도 → 공동인증 → 출금 |
| 4 | 직전송 코인 충전 | 외부 지갑 USDT/USDC → 환불주소 등록 → 수신주소 제공 → Triple-A API 환전 |
| 5 | 은행계좌 KRW 충전 | 등록 계좌 선택 → 금액 입력 → PIN 인증 → Beple Money 적립 |
| 6 | QR 결제 | 가맹점 QR 스캔 → 금액 확인 → PIN/생체인증 → 결제 완료 |
| 7 | ATM 현금 출금 | ATM QR 스캔 → 출금 금액 입력 (1만~50만) → PIN 인증 → 현금 수령 |
| 8 | 홈 대시보드 | Beple Money(원화) + Korbit 코인 자산 + 퀵액션 + 배너 |
| 9 | 계좌 관리 | ARS 인증으로 계좌 등록·삭제, 등록 계좌 목록 조회 |
| 10 | 거래 내역 | 날짜·결제수단·상태별 필터, 상세 내역 조회 |

## 6. 제약 사항

- KYC Quick 충전 한도: 50만원 (KYC Full: TBD)
- ATM 출금 범위: 1만원 ~ 50만원
- Korbit: 개인 지갑 주소 미지원 (법적 제약)
- 직전송 환불: 환불주소만 지원 (거래소 입금 주소 불가)
- Beple Money 기존 화면: WebView 재사용 (신규 개발 최소화)
- 생체인증·공동인증·출금동의: 별도 개발 필요
- 해외 계좌 등록: 미확정
